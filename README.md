# 🌍 Hope Foundation – NGO Charity Web Application

## 📌 Project Overview
Hope Foundation is a web-based NGO charity platform designed to facilitate online donations and volunteer registration. The system enhances user engagement by providing real-time feedback, donation receipts, and seamless interaction.

---

## 🚀 Features

### 💰 Donation System
- Online donation form
- Secure data storage using PHP & MySQL
- Auto-generated transaction ID
- Dynamic donation receipt
- Downloadable receipt

### 🙋 Volunteer Registration
- Volunteer sign-up form
- Stores user details in database
- Improves user engagement

### 📊 User Experience
- Real-time success notifications
- Modal-based receipt display
- Responsive UI design

---

---

## 🔄 System Workflow

1. User fills donation/volunteer form
2. JavaScript sends data using AJAX (Fetch API)
3. PHP processes the request
4. Data stored in MySQL database
5. Success response returned to frontend
6. UI updates with confirmation & receipt
